# 行程表功能測試清單

每次修改行程表相關程式碼後，請測試以下功能：

## 封面設定
- [ ] 上傳封面照片 → 照片顯示在預覽區
- [ ] 選擇城市預設圖片 → 照片顯示在預覽區
- [ ] 調整圖片位置 → 位置正確儲存
- [ ] 切換封面風格 → 預覽正確更新

## 特色行程
- [ ] 新增特色 → 顯示在列表
- [ ] 編輯特色標題/描述 → 正確儲存
- [ ] 刪除特色 → 從列表移除
- [ ] 重新整理頁面 → 特色還在

## 每日行程
- [ ] 新增天數 → 顯示新的一天
- [ ] 編輯標題/描述 → 正確儲存
- [ ] 新增活動 → 顯示在該天
- [ ] 刪除天數 → 天數編號自動調整

## 飯店資訊
- [ ] 新增飯店 → 顯示在列表
- [ ] 上傳飯店圖片 → 圖片顯示
- [ ] 單一飯店 → 應該置中顯示

## 自動存檔
- [ ] 修改任何欄位 → 30秒後顯示「已儲存」
- [ ] 重新整理頁面 → 資料還在

## 預覽
- [ ] 切換桌面/手機版 → 預覽正確切換
- [ ] 所有修改 → 即時反映在預覽

---

## 如何使用

1. 修改程式碼後，打開行程表編輯頁面
2. 依照清單逐項測試
3. 發現問題立即記錄並修復
4. 全部通過後才部署
