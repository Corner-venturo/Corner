## 🎯 獨特性等級說明

### Ultra Exclusive（極致獨家）

- 全球<10個名額/年
- 需特殊關係或極提前預約
- 例如：數寄屋橋次郎

### Highly Exclusive（高度獨家）

- 需提前6個月預約
- 嚴格人數限制
- 例如：祇園藝伎茶宴、旗袍訂製

### Exclusive（獨家）

- 限定名額
- 需提前1-3個月預約
- 例如：雪地晚宴、K-Pop課程

### Premium（精品）

- 高品質但較易預約
- 適合大多數高端客戶

## 🔄 如何查詢資料

### SQL 查詢範例

```sql
-- 查詢所有米其林三星餐廳
SELECT name, city_id, chef_name, avg_price_dinner, currency
FROM michelin_restaurants
WHERE michelin_stars = 3
ORDER BY country_id;

-- 查詢適合蜜月的體驗
SELECT name, city_id, price_per_person_min, currency
FROM premium_experiences
WHERE 'honeymoon' = ANY(recommended_for)
AND is_active = TRUE;

-- 查詢佣金率最高的項目
SELECT name, category, commission_rate
FROM premium_experiences
WHERE commission_rate >= 15
ORDER BY commission_rate DESC;

-- 查詢日本的所有頂級體驗
SELECT
  pe.name,
  c.name AS city,
  pe.exclusivity_level,
  pe.price_per_person_min,
  pe.currency
FROM premium_experiences pe
JOIN cities c ON pe.city_id = c.id
JOIN countries co ON pe.country_id = co.id
WHERE co.code = 'JP'
AND pe.is_active = TRUE;
```
