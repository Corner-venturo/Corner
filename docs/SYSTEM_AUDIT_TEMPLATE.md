| 歸還 | collected/rejected | 更新為 returned | `ReturnDocumentsDialog` |
| 退件 | pending/submitted/collected | 更新為 rejected | `BatchRejectDialog` |
| 編輯 | 行內 | 編輯簽證 | `EditVisaDialog` |
| 刪除 | 行內 | 刪除簽證 | 確認 Dialog |

#### 簽證狀態流轉

```
pending（待送件）
  ├─ [送件] → submitted（已送件）
  └─ [退件] → rejected（退件）

submitted（已送件）
  ├─ [取件] → collected（已取件）
  └─ [退件] → rejected（退件）

collected（已取件）
  ├─ [歸還] → returned（已歸還）終態
  └─ [退件] → rejected（退件）

rejected（退件）
  ├─ [送件] → submitted（重新送件）
  └─ [歸還] → returned（已歸還）終態
```
